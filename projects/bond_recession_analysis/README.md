# Bond Recession Analysis

This project compares U.S. bond yield data with official recession periods to analyze how the yield curve inversion precedes recessions.

## How to Run
```bash
python bond_recession_analysis.py
```
## Description
Loads DGS2 (2-Year), DGS10 (10-Year), and USREC (Recession indicator) data from FRED

Plots yield spreads and shaded recession areas

Demonstrates the "inversion â†’ recession" relationship
